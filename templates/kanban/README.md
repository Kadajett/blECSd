# Kanban Board Template

> **Status:** Placeholder - implementation coming soon

A kanban board demonstrating blECSd's layout system, drag-and-drop, and state management.

## Planned Features

- Multiple columns (To Do, In Progress, Done)
- Card creation, editing, and deletion
- Drag-and-drop cards between columns using keyboard
- Persistent state via JSON file
- Color-coded labels and priority levels
